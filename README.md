# lightweight-sedentary-reminder README

vscode 插件，轻量级的久坐提醒

> 有些快捷键不同，避免和 vscode 冲突

## usage

1. <kbd>ctrl/cmd</kbd> + <kbd>shift</kbd> + <kbd>p</kbd>
2. 输入 startWork 开始，输入 stopWork 退出

## Features (功能)

1. 默认的每 30min 提醒
2. 配置项 - 提醒时间

## Settings

| 参数                | 说明             | 类型   | 默认值 |
| ------------------- | ---------------- | ------ | ------ |
| `reminder.interval` | 提醒间隔（分钟） | number | 30     |
